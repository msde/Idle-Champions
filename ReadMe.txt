A work in progress gem farming script for the game Idle Champions. The script supplements Modron Automation to use Fkeys to level champions after a reset and farm Steelbones Stacks on Briv prior to a reset.

Warning: Reads from memory. I do not know CNE's stance on reading game memory, so use at your own risk. Pointers may break on any given update and I may no longer decide to update them.
